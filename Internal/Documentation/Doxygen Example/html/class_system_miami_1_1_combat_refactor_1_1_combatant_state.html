void SystemMiami.CombatRefactor.CombatantState.SwitchState ( CombatantState newState,
float delay )

An overload of SwitchState that takes a float. This function will run a Coroutine that then waits float delay seconds before assigning a new state and calling OnEnter() on it.

Parameters
newStateThe state to transition to.
Parameters
delayAmount of time in seconds for the state machine to wait between: (a) calling OnExit() on the previous state, and (b) calling OnEnter() on the new current state.
